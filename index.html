Afinal, o que é isso?

Archidata é uma ferramenta para a modelagem e edição de banco de dados através do uso de notações gráficas, como por exemplo, diagramas de modelagem. Archidata tem como objetivo fornecer uma plataforma extensível que permita aos desenvolvedores criar novos conteúdos que venham a extender suas funcionalidades através da colaboração da comunidade open-source.

Algumas características

Totalmente Gratuito

Além de gratuito o Archidata também é código aberto, assim você pode contribuir para melhorá-lo.

Vários bancos de dados

Archidata permite adicionar suporte a novos bancos de dados através de plug-ins!

Código SQL

Desenhe diagramas e gere o código SQL para cada banco de dados suportado pelo software.

Engenharia Reversa

Obtenha automaticamente um diagrama a partir de um banco de dados já existente!

Plug-ins

Você pode estender as funcionalidades do software através de novos plug-ins!

Exportar

Você pode exportar diagramas para vários formatos como bmp, jpg, png, xps e outros!

Customizável

Você pode customizar diagramas com cores e estilos para dar personalidade aos seus diagramas!

Feito com amor

Dedicamos muito amor, suor e café para produzir uma ferramentado qual nos orgulhamos. Aproveite!
